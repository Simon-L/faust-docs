Detailed Description

Utility class for dispatching MIDI data to a list of android.media.midi.MidiReceivers. This class subclasses android.media.midi.MidiReceiver and dispatches any data it receives to its receiver list. Any receivers that throw an exception upon receiving data will be automatically removed from the receiver list, but no IOException will be returned from the dispatcher's android.media.midi.MidiReceiver#onReceive in that case.
