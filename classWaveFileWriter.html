Detailed Description

Write audio data to a WAV file.


WaveFileWriter writer = new WaveFileWriter(waveFileOutputStream);
writer.setFrameRate(48000);
writer.setBitsPerSample(24);
writer.write(floatArray, 0, numSamples);
writer.close();
