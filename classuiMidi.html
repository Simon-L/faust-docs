Detailed Description

uiMidi : Faust User Interface This class decodes MIDI meta data and maps incoming MIDI messages to them. Currently ctrlChange, keyOn/keyOff, keyPress, progChange, chanPress, pitchWheel/pitchBend start/stop/clock meta data is handled. MIDI channel is numbered in [1..16] in this layer. Channel 0 means "all channels" when receiving or sending.
