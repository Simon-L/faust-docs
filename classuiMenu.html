Detailed Description

Intern class for Menu

A popup menu. The names and values used for the menu items are described in the string mdescr with the following syntax "{'foo':3.14; 'faa':-0.34; ... 'fii':10.5}"

Constructor.

Initialize the uiComponent and Menu specific variables, and the juce::ComboBox parameters. Menu is considered as a slider in the FAUST logic, with a step of one. The first item would be 0 on a slider, the second 1, etc. Each "slider value" is associated with a string.

Parameters
gui,zone,w,h,tooltip,labeluiComponent variables.
curCurrent "slider value" associated with the current item selected.
lowLowest value possible.
hiHighest value possible.
mdescrMenu description. Contains the names of the items associated with their "value".
