Detailed Description

a faust root node

A Faust root node handles the 'hello' message and provides support for incoming osc signal data.

virtual void oscfaust::RootNode::get ( unsigned long  ipdest,
const std::string &  what 
) const
virtual

handler for the 'get' 'attribute' message

Parameters
ipdestthe output message destination IP
whatthe requested attribute

The 'get' message is supported by every node:

  • it is propagated to the subnodes until it reaches terminal nodes
  • a terminal node send its state on 'get' request to the IP address given as parameter. The get method is basically called by the accept method.
