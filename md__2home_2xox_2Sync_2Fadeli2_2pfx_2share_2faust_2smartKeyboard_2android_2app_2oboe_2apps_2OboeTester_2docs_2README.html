OboeTester Documentation

OboeTester is an app that can be used to test many of the features of Oboe, AAudio and OpenSL ES. It can also be used to test Android devices, measure latency and glitches, etc.

OboeTester can be run as an interactive app. It can also be run as part of an automated test using Intents

Install OboeTester

You have two options: 1) Download OboeTester from Play Store 2) OR Build latest OboeTester using Android Studio
