Detailed Description

MIDI processor definition.

MIDI input or output handling classes will implement this interface, so the same method names (keyOn, keyOff, ctrlChange...) will be used either when decoding MIDI input or encoding MIDI output events. MIDI channel is numbered in [0..15] in this layer.
