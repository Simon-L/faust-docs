Detailed Description

Plotting style, used for both layout and SVG rendering. Colour/dash/hatch styles are defined as CSS classes, assigned to elements based on their integer style index. CSS is written inline in the SVG, and can be extended/overridden with .cssPrefix/.cssSuffix.

It generates CSS classes from .colours (svg-plot-sN/svg-plot-fN/svg-plot-tN for stroke/fill/text), .dashes (svg-plot-dN) and .hatches (svg-plot-hN), where N is the index - e.g. there are six colours by default, generating svg-plot-s0 to svg-plot-s5.
